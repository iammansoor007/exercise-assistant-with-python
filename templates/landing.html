University Project Overview

This AI Fitness Assistant represents my final year computer science project, demonstrating the application of modern AI technologies to solve real-world problems in the fitness industry.

  • Developed over 6 months as a final year project
  • Implements pose estimation using TensorFlow.js and MediaPipe
  • Features real-time exercise form analysis
  • Includes comprehensive performance tracking
  • Built with Python Flask backend and modern web technologies
